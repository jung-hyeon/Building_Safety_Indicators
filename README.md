## ✍ 요약
본 연구는 건물 화재에 대한 화재 피해(인명피해, 재산피해)를 예측하는 모델을 개발하여 화재 위험성 지표(화재피해등급)를 산출하고자 한다.

해당 지표를 활용해 건물 화재 위험도를 실시간으로 분석하고 선제적인 대응전략을 수립하여 화재 발생에 의한 피해를 최소화 할 수 있을 것으로 기대한다.
